    FAQ

    Quick questions.

    Is this available right now?

    It depends on where you are, how many batteries are out, and road conditions. Send the form or text us and we will check.

    What if the outage lasts several days?

    The $199 rental includes up to four extra battery swaps during the same outage.

    Can this run my whole house?

    No. This is for essentials like a fridge, freezer, router, phones, lights, CPAP, and small devices.

    Can I use it indoors?

    Yes. The batteries do not create exhaust. Please follow the device instructions.

    Will it run my CPAP?

    Usually, yes. Runtime depends on your machine and whether you use heat or humidifier settings.

    Will it run an oxygen concentrator?

    Maybe. We need the exact model first so we can check the power draw.

    Do you offer delivery?

    Yes. Delivery, help plugging in, swaps, and pickup are included in the $199 rental when service is available.

    Do I need extension cords?

    One extension cord is included. We may have extra cords or power strips available if the setup calls for it.
